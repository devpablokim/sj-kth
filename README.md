# sj-kth — (주)신정개발 경영지원 대시보드

엑셀 파일을 데이터 원본으로 사용하는 경량 경영지원 관리 대시보드입니다.
매출·매입·프로젝트·경비·인사급여·장비차량·자금계좌 현황을 한 화면에서 조회할 수 있습니다.

## 🔒 보안 원칙 (중요)

**회사 데이터는 절대 이 저장소에 올라가지 않습니다.**

- 엑셀 파일은 사용자가 브라우저에서 직접 선택하며, **100% 브라우저 안에서만 파싱**됩니다. 서버로 전송되는 데이터가 없습니다 (API 호출 자체가 없음).
- 불러온 데이터는 사용자 브라우저의 localStorage에만 저장되며, "데이터 지우기" 버튼으로 즉시 삭제할 수 있습니다.
- `.gitignore`가 모든 스프레드시트 파일(`*.xlsx`, `*.xls`, `*.csv` 등)과 `/data/` 폴더의 커밋을 차단합니다. **이 규칙을 절대 해제하지 마세요.**
- 저장소에 포함된 `lib/sampleData.json`은 화면 데모용 가상 데이터로, 실제 기업 정보가 아닙니다.

## 사용 방법

1. 배포된 대시보드 주소에 접속
2. 경영지원 마스터 엑셀 파일(내 컴퓨터에 보관)을 드래그하거나 선택
3. 개요·매출·매입·프로젝트 등 탭으로 조회
4. 엑셀을 수정한 뒤에는 "다른 파일 불러오기"로 다시 선택하면 최신 데이터로 갱신

### 마스터 엑셀 양식

- 필수 시트: `매출`, `매입`, `프로젝트` / 선택 시트: `경비`, `인사급여`, `장비차량`, `자금계좌`, `거래처`
- 모든 데이터 시트는 **1행 헤더 + 2행부터 데이터**, 셀 병합 없음
- 자세한 입력 규칙은 마스터 엑셀 안의 **가이드** 시트 참고

## 개발

```bash
npm install
npm run dev   # http://localhost:3000
npm run build # 프로덕션 빌드
```

기술 스택: Next.js (App Router) · TypeScript · Tailwind CSS · SheetJS(xlsx) · Recharts

## 배포 (Vercel)

1. [vercel.com](https://vercel.com) → Add New Project → 이 GitHub 저장소(import `devpablokim/sj-kth`) 선택
2. 프레임워크가 Next.js로 자동 감지되므로 설정 변경 없이 Deploy
3. 이후 `main` 브랜치에 push할 때마다 자동 재배포
